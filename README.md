# React Portfolio (Lovable-friendly)

Simple one-page portfolio built with React + Vite (no Next.js-specific libraries).

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Why this setup

- Uses plain React + Vite so it stays lightweight and easy to sync.
- Avoids Next.js routing, server features, and Next-only packages.
- Uses standard browser-friendly CSS and React components.

You can customize text/content in `src/App.jsx` and styling in `src/styles.css`.
